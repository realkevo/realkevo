### About Me

I help businesses become more efficient and reliable using software. From **automation workflows** to **mobile apps** and **web systems**, I focus on solving actual problems:  

- **Automation & Workflow** – I build scripts and tools that reduce repetitive tasks, handle data pipelines, and ensure systems update automatically.

- **Web Development** – I develop websites and dashboards that integrate with existing business systems, making data accessible and actionable.
